## 🚀 加载扩展步骤

### 第一步：打开Chrome扩展管理页面
1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 按回车键

### 第二步：启用开发者模式
1. 在扩展管理页面右上角
2. 找到"开发者模式"开关
3. 点击开关，启用开发者模式

### 第三步：加载扩展
1. 点击"加载已解压的扩展程序"按钮
2. 在文件选择对话框中，导航到：
   ```
   /Volumes/张少雄/AI智能阅卷助手V5/ai-grading-extension
   ```
3. 点击"选择文件夹"或"确定"

### 第四步：验证扩展加载成功
扩展加载成功后，您应该看到：
- 扩展列表中出现"智学网AI智能阅卷助手"
- 扩展状态为"已启用"
- 工具栏中出现扩展图标（可能是默认图标）

## 🛠️ 故障排除

### 问题1：扩展无法加载

**症状**：
- 提示"无法加载扩展程序"
- 扩展状态显示"已损坏"

**解决方案**：
1. 检查文件夹路径是否正确
   ```
   /Volumes/张少雄/AI智能阅卷助手V5/ai-grading-extension
   ```
2. 确认manifest.json文件存在
3. 运行验证脚本：
   ```bash
   node verify-extension.js
   ```

### 问题2：扩展已加载但无图标

**症状**：
- 扩展已启用但工具栏无图标

**解决方案**：
- 这是正常现象，因为我们移除了图标引用
- 扩展功能不受影响
- 您仍然可以通过快捷键或右键菜单使用

### 问题3：扩展加载后立即报错

**症状**：
- 扩展显示"出现错误"
- Console显示JavaScript错误

**解决方案**：
1. 刷新智学网页面
2. 检查浏览器Console（按F12）
3. 确认网络连接正常
4. 验证API密钥配置

### 问题4：popup页面无法打开

**症状**：
- 点击扩展图标无反应
- 或显示空白页面

**解决方案**：
1. 检查popup.html文件是否存在
2. 确认没有JavaScript语法错误
3. 检查浏览器Console中的错误信息
4. 重新加载扩展

## 📊 扩展权限说明

扩展需要以下权限来正常工作：

### 必需权限
- **storage** - 保存用户设置和评分记录
- **activeTab** - 访问当前智学网页面
- **scripting** - 注入评分功能到页面

### 可选权限
- **unlimitedStorage** - 存储大量评分数据
- **downloads** - 导出评分报告

### 主机权限
- **zhixue.com** - 访问智学网平台
- **CDN域名** - 加载外部库（Tailwind CSS, Chart.js等）

## 🎨 重新添加图标（可选）

如果您想为扩展添加图标，可以：

### 方法1：使用SVG图标
1. SVG图标已创建：`assets/icons/icon.svg`
2. 使用在线工具转换为PNG：
   - https://cloudconvert.com/svg-to-png
3. 生成16x16, 32x32, 48x48, 128x128四个尺寸
4. 保存到 `assets/icons/` 目录
5. 更新 `manifest.json` 恢复图标引用

### 方法2：使用默认图标
1. 从Chrome扩展示例中复制图标
2. 或使用任何128x128的PNG图片
